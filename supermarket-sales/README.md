# Dashboard Sales Supermarket

Este é um projeto de um Dashboard que faz a leitura de um .CSV para análise de vendas de uma rede de Supermercados.

## Run

```shell
streamlit run dashboards.py
```

## 👨‍💻Tecnologias:
> - **PYTHON**
> - Pandas
> - Plotly
> - streamlit
